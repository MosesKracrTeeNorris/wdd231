2. Site Purpose

PENIEL is an IT service provider website. Focused on showcasing training, sales of electronic devices and computer accessories, busic computer training skills. The site provides:

  • Home Page (index.html): Introduction of PENIEL with hero section
  • Course Page (course.html): Displays all course items dynamically loaded from a JSON data source with category filtering and search functionality
  • About Us Page (about.html): This will be a brief story about our business journey. Services we provide, and mission and vision statements.
  • Contact Page (contact.html): Contact form for customer inquiries and business information will be provided
  • Thank You Page (thankyou.html): Form submission confirmation page will be provided

Dynamic Features

This site will demonstrates JavaScript proficiency through:

  • Data fetching from JSON file using Fetch API with try/catch error handling
  • Dynamic content generation displaying items with properties each.
  • Local storage for wishlist/favorites functionality
  • Modal dialogs for detailed sales and services information
  • DOM manipulation and event handling (clicks, filters, sorting)
  • Array methods (filter, map, forEach) for data processing
  • Template literals for dynamic HTML generation
  • ES Modules for code organization

3. Target Audience Scenarios

Scenario 1:

Viewing of the latest electronic devices with their prices before making a purchase decision.

Answer: Browse the list on the course page where all items or courses will be displayed with detailed descriptions, and prices.

Scenario 2:

Viewing of saved items while navigating courses or services.

Answer: Click the heart icon on any course or service of your choice to add it to your wishlist. Your selections will be saved using local storage and be displayed throughout the browser session.

Scenario 3:

View and learn about PENIEL Services, and contact the business about Sales or Services you want.

Answer: Visit the About Us page to learn more about PENIEL's business journey and experience. Then use the contact form that will be provided on the Contact page to submit inquiries. Also find direct contact information including email and phone number.
